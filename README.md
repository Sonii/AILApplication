READEME :

Look-at est un site communautaire de partage de site. Je connais un bon site et je veux en faire profiter les autres.

Utilisation :

-En mode déconnecté et connecté
Naviguer dans les catégories (Sport par exemple ;)
Regarder les sites qui vous intéresse => cela fait monter le nombre de vue
Visionner les commentaires
Créer un compte
Se connecter
-En mode connecté seulement
Ajouter/Supprimer un site
Ajouter/Modifier/Supprimer un commentaire
Se deconnecté
Fonctionnalité :

De se loguer (Système d’authentification).
De publier leur Site Web.
De publier leurs commentaires.
De supprimer leurs Sites Web de l’application.
De supprimer leurs commentaires.
De Modifier leurs commentaires.
Chaque catégorie répertorie un classement des sites les plus visités


La publication d’un site Web ou d’un commentaire nécessitent à son auteur d’être logué. 
Tous les utilisateurs ont le droit de lire et de publier un commentaire sur les sites qui leur 
appartiennent, ainsi que les sites qui ne leur appartiennent pas.

Un site web ne peut être supprimé que par la personne qu’elle l’a publiée. Une vérification côté 
serveur est mise en place pour vérifier que la personne qui tente de supprimer le site en question a 
bien le droit de le faire.

Bien que les options supprimer et modifier un commentaire soient présentes sur tous les 
commentaires. Un commentaire ne peut être supprimé ou modifié que par la personne qu’elle 
l’a publiée. Quand une personne essaye de cliquer sur une option supprimer ou modifier un 
commentaire qui ne lui appartient pas une page d’erreur est envoyé au client.

Une vérification côté serveur est mise en place pour vérifier si la personne en question a le droit de 
supprimer ou modifier le commentaire en question.

Pour supprimer un commentaire ou un site web la personne est tenue de confirmer son mot de 
passe.


Choix d'Implémentation :

Gestion des accents mySQL
Architecture spring MVC avec JDBC
Utilisation des EL et de la JSTL
Utilisation des variables de sessions
Utilisation de JavaScript (verification de formulaire coté client) et de CSS
Catégories choisis et statiques pour le projet (démonstration)
